
# 💧 Controle de Nível de Água

Sistema de monitoramento de reservatório desenvolvido em Python, com exibição em tempo real no terminal.

## 🚀 Funcionalidades
- Simulação automática do nível da água
- Classificação por níveis (crítico até máximo)
- Exibição colorida com colorama
- Barra visual de progresso
- Alertas de nível crítico e máximo
- Atualização contínua em tempo real
